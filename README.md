# Task Management API

## Features
- User Authentication
- Task CRUD Operations
- Docker Support
- Jenkins CI/CD Pipeline
- SonarQube Integration
- Automated Testing with Jest

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Docker
- Jenkins
- SonarQube
- Jest

## Installation

npm install

## Run Application

npm start

## Docker Commands

docker build -t task-api .
docker run -p 3000:3000 --env-file .env task-api

## Jenkins Pipeline Stages
- Build
- Test
- Code Quality
- Security Scan
- Deploy
